# Neon Snake - 霓虹贪吃蛇

基于 GitHub 项目 [nullxjx/Snake](https://github.com/nullxjx/Snake) 设计的前端版本，使用 React + TypeScript + Canvas 开发。

## 功能特性

### 核心玩法
- 🐍 经典贪吃蛇玩法，支持方向键或 WASD 控制
- 🤖 AI 蛇对战模式，AI 使用 A* 寻路算法自动寻找食物
- 🧱 障碍物系统，可以用子弹击毁
- 🔫 子弹系统，按 Shift 发射，吃 🔫 食物补充弹药

### 食物系统
| 食物 | 表情 | 效果 |
|------|------|------|
| 普通食物 | 🍎 | +1分，长度+1 |
| 高级食物 | 🍋 | +3分，长度+1 |
| 弹药包 | 🔫 | 子弹+3，长度+1 |

### 游戏模式
1. **经典模式** - 传统贪吃蛇玩法
2. **挑战模式** - 加入障碍物和子弹系统
3. **对战AI** - 与 AI 蛇竞争食物

### AI 寻路策略
```
AI.move() {
  1. 获取头部坐标
  2. 获取所有食物位置
  3. 寻找安全路径（吃完食物后仍能回到尾巴）
     - 存在：选择最短路径
     - 不存在：选择最长路径前往尾巴
     - 仍不存在：死亡
}
```

### 碰撞规则
- 玩家蛇撞墙/撞自己 → 游戏结束
- 玩家蛇撞 AI 蛇头部 → 游戏结束
- AI 蛇撞玩家蛇 → AI 可通行
- 子弹击中障碍物 → 障碍物消失

## 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite
- **状态管理**: Zustand
- **样式**: Tailwind CSS
- **渲染**: HTML5 Canvas

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 操作说明

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 控制蛇移动 |
| Shift | 发射子弹 |
| 空格 | 暂停/继续 |
| ESC | 重新开始 |
